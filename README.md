# Real Estate Site

Site for searching Real Estate advermisements

# Set up database

Creating new base
```bash
python database_manager.py --newbase --loaddata ads.json
```
Add new data to the base

```bash
python database_manager.py --loaddata new_ads.json
```

# Running site

```bash
$python server.py
```
